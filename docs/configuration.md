# ⚙️ 서버 설정 및 구조 (Server Configuration)

**Slave**를 서버에 초대한 후, 원활한 관리를 위해 아래 설정을 진행해 주세요.

---

## 1. 채널 설정 명령어 (Admin Only)
채팅창에 아래 명령어를 입력하여 각 로그 채널을 지정합니다.
* `!set log [#채널]` : 메시지 수정/삭제 등 일반 활동 로그 채널 지정
* `!set punish [#채널]` : 뮤트, 타임아웃, 차단 등 제재 내역 로그 채널 지정
* `!set bot [#채널]` : 봇 명령어 사용이 허용되는 전용 채널 지정
* `!set ticket [#채널]`: 티켓 생성 알림 및 문의가 생성될 위치 지정


## 2. 설정 확인
설정된 값은 서버 내부 데이터베이스(`config.json`)에 안전하게 저장되며, 봇 재시작 후에도 유지됩니다.
