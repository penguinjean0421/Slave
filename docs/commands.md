# 📜 명령어 요약 (Command Summary)

Slave 봇이 제공하는 주요 기능과 명령어 사용법에 대한 가이드입니다.

---

## ✨ 주요 기능 (Key Features)

### 1. 🛠️ 서버 관리 (Moderation)
* **음성 채널 제어**: 유저 뮤트/데픈(시간 설정 가능), 음성 채널 강제 퇴장(vckick).
* **유저 제재**: 타임아웃(채팅 금지), 추방(Kick), 차단(Ban/Unban).
* **스마트 파싱**: `10s`, `5m`, `1h`, `1d` 등 직관적인 시간 단위 지원.

### 2. 📝 실시간 로그 시스템 (Logging)
* **활동 로그**: 멤버 입퇴장, 메시지 수정/삭제 내용 실시간 추적.
* **음성 로그**: 채널 입장/퇴장 및 이동 경로 기록.
* **처벌 전용 로그**: `punish` 채널을 별도로 지정하여 관리 내역만 따로 보관 가능.

### 3. 🎫 문의 지원 (Ticket System)
* 하이브리드 생성: !ticket 명령어 또는 버튼 클릭 패널을 통해 1:1 문의 채널을 생성합니다.
* 프라이빗 상담: 관리자와 요청자만 접근 가능한 독립된 채널을 자동 생성하여 보안을 유지합니다.
* 간편 종료: 상담 완료 후 명령어를 통해 채널을 즉시 정리할 수 있습니다.

### 4. ⚙️ 설정 (System)
* **자동 구성**: 봇 입장 시 서버 정보를 `config.json`에 자동 등록 및 갱신.
* **접두사 관리**: `.env`에서 설정한 여러 개의 접두사(Prefix)를 동시에 지원.

### 5. 🔗 유틸리티 & 정보 (Utility)
* **Github 검색**: 등록된 프로젝트나 프로필로 바로가는 링크 제공.
* **결정 도우미**: `choose` 명령어로 여러 선택지 중 하나를 무작위 선택.
* **메뉴 추천**: 상황(아침/점심/저녁/야식)이나 종류(한식/중식 등)별 메뉴 추천.

## 📋 명령어 목록

### 유저 명령어

#### 정보 및 서버 문의
| 명령어 | 설명 | 예시 |
| :---: | :---: | :---: |
| `!help` | 전체 명령어 가이드 확인 | `!help`, `!help 관리자` |
| `!github` | 등록된 깃허브 링크 출력 | `!github 과제` |
| `!credit` | 개발자 정보 확인 | `!credit` |
|`!open` | 티켓 생성 | `!open`|
|`!close` | 티켓 종료 | `!close`|

#### 게임 전적조회
| 명령어 | 설명 | 예시 |
| :---: | :---: | :---: |
| `!lol` | LOL 전적조회 | `!lol 국가 닉네임#테그` |
| `!pubg` | PUBG 전적조회 | `!pubg 플랫폼 닉네임` |

#### 유틸리티
| 명령어 | 설명 | 예시 |
| :---: | :---: | :---: |
| `!choose` | 선택지 중 하나를 무작위 선택 | `!choose 자장면 짬뽕` |
| `!menu` | 상황/종류별 메뉴 추천 | `!menu 점심`, `!menu 일식` |
| `!kill` | 마인크래프트식 다잉메세지 출력 | `!kill`, `!kill @유저` |

### 관리자 명령어

#### 티켓 관리
| 명령어 | 설명 | 예시 |
| :---: | :---: | :---: |
|`!answer`| 티켓 답장 | `!answer 내용` |

#### 음성 제재
| 명령어 | 설명 | 예시 |
| :---: | :---: | :---: |
| `!mute` | 마이크 차단(시간제 가능) | `!mute @유저 10m` |
| `!unmute` | 마이크 차단 해제 | `!unmute @유저` |
| `!deafen` | 헤드셋 차단 | `!deafen @유저 10m`|
| `!undeafen` | 헤드셋 차단해제 | `!undeafen @유저` |
| `!vckick` | 음성 채널 강제 퇴장 | `!vckick [유저] (사유)`|

#### 서버 재제
| 명령어 | 설명 | 예시 |
| :---: | :---: | :---: |
| `!timeout` | 타임아웃 적용 | `!timeout @유저 1h 사유` | 
| `!kick` | 서버 추방 | `!kick [유저] (사유)` |
| `!ban` | 서버 차단 | `!ban [유저] (사유)` |
| `!unban` | 차단 해제 | `!unban [ID/닉네임]` |

#### 채널 설정
| 명령어 | 설명 | 예시 |
| :---: | :---: | :---: |
| `!set` | 채널 설정 | `!set [log/punish/bot/ticket] [#채널]` |
|`!reset` | 설정 해제 | `!reset [log/punish/bot/ticket/all]` |
